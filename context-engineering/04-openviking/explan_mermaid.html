OpenClaw 对话生命周期（Mermaid 时序图）

sequenceDiagram
      autonumber
      participant U as User
      participant G as Gateway
      participant CE as ContextEngine
(lossless-claw)
      participant M as Memory
(memory-openviking)
      participant A as Agent

      U->>G: 发送新消息

      Note left of CE: 会话初始化: bootstrap() 恢复状态

      rect rgb(240, 248, 255)
          Note over G, CE: [Phase 1: 上下文摄入与组装]
          G->>CE: ingest
          Note right of CE: 消息持久化、标签化、重要性分析

          G->>CE: assemble
          CE-->>G: 返回当前窗口上下文 (含摘要/历史)
      end

      rect rgb(245, 245, 220)
          Note over G, M: [Phase 2: 记忆增强]
          opt 触发长时记忆检索
              G->>M: before_agent_start
              Note right of M: 基于当前 Query 进行向量检索
              M-->>G: 返回召回的长期记忆片段
          end
      end

      rect rgb(240, 255, 240)
          Note over G, A: [Phase 3: 模型执行]
          G->>A: 调用 Agent
          Note right of A: Prompt = 长期记忆 + 摘要 + 最近历史
          A-->>G: 返回响应结果
      end

      rect rgb(255, 245, 238)
          Note over G, CE: [Phase 4: 上下文维护与压缩]
          A->>CE: afterTurn
          Note right of CE: 统计更新、DAG 节点挂载

          alt 上下文超限
              G->>CE: compact
              Note right of CE: 基于 DAG 的无损压缩/摘要化
          end

          alt 涉及子任务
              G->>CE: prepareSubagentSpawn
              A->>A: 子代理执行 (独立 Scope)
              A->>CE: onSubagentEnded
              Note right of CE: 子代理结果归并至主会话
          end
      end

      rect rgb(255, 240, 245)
          Note over G, M: [Phase 5: 记忆捕获]
          opt 对话成功结束
              G->>M: agent_end
              Note right of M: 提取关键信息存入向量库
              M-->>G: 记忆存储完成
          end
      end

      G->>U: 返回最终响应
